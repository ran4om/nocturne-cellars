# Nocturne Cellars

A one-page cinematic winery concept for a fictional small-production Etna estate.

## Run locally

```bash
npm install
npm run dev
```

Build and preview:

```bash
npm run build
npm run preview
```

## Implementation notes

- React and Vite.
- Scroll-driven sticky film sequence using passive listeners, refs, and `requestAnimationFrame`.
- Full stacked-content fallback for reduced motion and small screens.
- Semantic DOM content and keyboard-visible focus states.
- Inquiry actions use fictional `.example` email addresses.
